<h1 align="center">🔥 Talkio - Real-time Chat Application</h1>

<p align="center">
A full-stack chat application built with <strong>MERN Stack</strong>, <strong>Zustand</strong>, <strong>Socket.IO</strong>, and <strong>Cloudinary</strong>, featuring:
</p>

<ul>
  <li>💬 Realtime messaging</li>
  <li>🖼️ Image sharing</li>
  <li>🔐 User authentication</li>
  <li>🎨 Theme toggle with DaisyUI</li>
</ul>
